# ЛР4 — Распознавание музыкальных жанров

## Описание проекта
Android-приложение для распознавания музыкальных жанров по аудиофайлу в формате WAV.

Приложение:
- позволяет выбрать аудиофайл с устройства;
- выполняет анализ аудиосигнала;
- определяет музыкальный жанр с помощью локальной модели машинного обучения;
- выводит результат распознавания на экран.

## Тема лабораторной работы
Машинное обучение на устройстве

## Вариант
Вариант 12 — «Распознавание музыкальных жанров по аудио»

## Используемые технологии
- Kotlin
- Android Studio
- TensorFlow Lite
- Librosa-based audio feature extraction
- WAV audio processing

## Поддерживаемые жанры
- Disco
- Classical
- Country
- Blues
- Metal
- Rock
- Reggae

## Структура проекта
- `app` — основное Android-приложение
- `core` — базовые компоненты проекта
- `genreclassifier` — логика классификации жанра
- `librosa` — обработка аудиосигнала и извлечение признаков

## Инструкция по запуску
1. Открыть проект в Android Studio.
2. Подключить Android-устройство.
3. Собрать и запустить приложение.
4. Выбрать аудиофайл `.wav`.
5. Нажать кнопку сканирования.
6. Получить результат распознавания жанра.

## Особенности
- Работа выполняется локально на устройстве.
- Для корректной работы используется формат `.wav`.
- Модель загружается из ресурсов приложения.

## Результат
Приложение успешно выполняет выбор WAV-файла, обработку аудиоданных и определение музыкального жанра.
